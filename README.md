## インストール

### ローカルにインストールする

* gem install faraday

### Gemfile に記述する

```
gem 'faraday'
```

```
$ bundle
```

## サンプルプログラム faraday-sample.rb 

フォークまたはクローンして、

```
$ bundle
```
を実行してください。

```
$ ruby faraday-sample.rb
```
で実行できます。
